RESTravelTiers API

About

This API can be used to get up to date information about the current English travel restictions put in place by the UK government which is linked here

How to use

Making requests

To make a requst is simple, just add the name of the country you want to retrive information from to the end of the url

www.WHATEVERTHEURLIS/API/<CountryName>

Input Params

Currently the only supported input parameter is the name of the country,

Output

Upon a sucessful request the you should recive JSON in the following format

Country { tier : "color" }
